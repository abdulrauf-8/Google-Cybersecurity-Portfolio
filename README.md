## Portfolio Projects

### 🔍 Security Audits
Conducted internal security audits using the NIST Cybersecurity Framework — identifying risks, evaluating controls, and producing compliance checklists.

### 🌐 Network Security Analysis
Analyzed network traffic, identified attack patterns (DoS, MITM, packet sniffing), and documented incident response procedures.

### 🐧 Linux & SQL
Applied Linux command-line skills for file permission management and used SQL to query and filter security logs.

### ⚠️ Threat & Vulnerability Assessment
Performed threat modeling, asset classification, and vulnerability assessments using industry frameworks (OWASP, NIST SP 800-30).

### 🚨 Detection & Incident Response
Worked with SIEM tools to detect threats, analyze alerts, and draft incident handler journals following the NIST Incident Response Lifecycle.

### 🐍 Python Automation
Wrote Python scripts to automate repetitive security tasks, parse log files, and update allow/deny lists.
